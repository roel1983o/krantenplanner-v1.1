
# Krantenplanner V1.1

Webinterface voor het uitvoeren van de krantenplanner pipeline.

## Start lokaal

```
pip install -r requirements.txt
streamlit run app/app.py
```

## Structuur

assets/ → vaste bestanden  
assets/templates/ → alle JPG templates  
notebooks/pipeline.ipynb → originele pipeline notebook  
app/ → web interface
